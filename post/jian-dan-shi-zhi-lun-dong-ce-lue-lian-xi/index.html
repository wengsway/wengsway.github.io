简单市值轮动策略练习

2018-04-21 8 min read # Finance

昨天学习了 JoinQuant 上 【新手入门教程】简单市值轮动策略 ，今天练习一下。

小市值策略

A股市场常常推崇小市值公司，因此小市值策略在A股市场会获得不错的收益，常见的持证逻辑有：

  • 小市值的上市公司中有很多成长性公司，股价上升有基础。

  • 小市值的上市公司容易“蹭概念”，更容易被炒作。

  • 小市值公司容易被借壳。

策略内容

每隔10个交易日，等金额买入市值最小的6只股票，卖出其他股票。

这里以A股的所有股票为例，运用上述策略。

初始化

按照策略的套路，先进行初始化设置。

def initialize(context):
    # 持有最小市值股票数
    g.stocksnum = 6
    # 轮动频率
    g.period = 10
    # 周期循环daily
    run_daily(daily,time='every_bar')
    # 记录策略进行到第几天，初始为1
    g.days = 1
    # 设定沪深300作为基准
    set_benchmark('000300.XSHG')
    # 使用动态复权（真实价格）
    set_option('use_real_price', True)
    # 设置买入时佣金万分之三，卖出时佣金万分之三加千分之一印花税, 每笔交易佣金最低扣5块钱
    set_order_cost(OrderCost(open_tax=0, close_tax=0.001,
                    open_commission=0.0003, close_commission=0.0003,
                    close_today_commission=0, min_commission=5), type='stock')

由于A股的股票可能会发生变化，所以在初始化这里不做设置，而是放在每天都会循环的 run_daily 函数中，保证成分股数据是最新的。

每日循环

每日循环部分要完成：选股、判断天数、进行买卖操作等步骤。

选股的流程可以简单概括为：确定股票池、获得股票池内每只股票的市值并从小到大排序、取出排在前6位的股票。

完整策略

def initialize(context):
    # 持有最小市值股票数
    g.stocksnum = 6
    # 轮动频率
    g.period = 10
    # 周期循环daily
    run_daily(daily,time='every_bar')
    # 记录策略进行到第几天，初始为1
    g.days = 1
    # 设定沪深300作为基准
    set_benchmark('000300.XSHG')
    # 使用动态复权（真实价格）
    set_option('use_real_price', True)
    # 设置买入时佣金万分之三，卖出时佣金万分之三加千分之一印花税, 每笔交易佣金最低扣5块钱
    set_order_cost(OrderCost(open_tax=0, close_tax=0.001,
                    open_commission=0.0003, close_commission=0.0003,
                    close_today_commission=0, min_commission=5), type='stock')
def daily(context):
    # 直接用 if 语句进行判断，每过一天，g.days 就加一天。
    if g.days % g.period == 1:
        # 获取当前时间
        date=context.current_dt.strftime("%Y-%m-%d")
        # 获取A股所有股票的代码
        stocks=get_index_stocks('000300.XSHG')+get_index_stocks('399106.XSHE')
        # 获得stocks内股票市值并排序
        df = get_fundamentals(query(valuation.code)
                                    # 获取市值数据表中的股票代码 
                                   .filter(valuation.code.in_(stocks))
                                            # 根据 stocks 中的股票进行筛选
                                           .order_by(valuation.market_cap.asc())
                                                      # 按照市值从小到大排序
                                                     ,date= date
                             )
        # 取出 df 中前g.stocksnum名的股票代码，放在 buylist 中
        buylist =list(df['code'][:g.stocksnum])
        # 对持有的股票进行判断：现在是否已经不在 buylist 里，如果是则卖出
        for stock in context.portfolio.positions:
            if stock not in buylist:
                #调整 stock 的持仓为0，即卖出
                order_target(stock, 0)
        # 将资金分成 g.stocksnum 份
        position_per_cash=context.portfolio.cash/g.stocksnum 
        # 用 position_per_cash 大小的 g.stocksnum 份资金去买buylist 中的股票
        for stock in buylist:
            order_value(stock, position_per_cash)
	else:
        pass # 什么也不做
	# 每执行一次，天数增加1
    g.days = g.days + 1

从图中超额收益线（黄色线）也可以看出，小市值轮动策略在2015—2017年上半年之间表现优异，而之后逐渐失去优势。

一些思考

小市值策略根本的盈利逻辑是“低买高卖”，近几年表现不错是因为上市公司最低市值从几亿飙升到几十亿，正好站在了风口上。

目前，A股市场上：

  • 市值小于20亿的股票：13只。
  • 市值位于20—50亿的股票：463只
  • 市值位于50—100亿的股票：419只

目前，市值最小的3只股票是：友好集团（600778，10.06%）、山东金泰（600385，1.01%）和ST仰帆（600421，-0.28%）

但由于近几年新股发行越来越快，小市值公司越来越多，而且用于回测的时间段正好是炒作壳价值的时间，这一策略的效果估计不会有现在这么好。

知乎用户量化师在一个回答中提到：

不难看出，小市值股票比例越高，长期收益也越高，这就是之前说的两个内因在起作用。但是，在小市值策略的收益来源中，即使我们以涨得最多的中证1000指数来对标，其版块上行带来的收益也就最多占了其中的三成，那剩下38.85%的年化收益其实就是轮动带来的超额收益。换句话说，只要轮动收益大于同时段内版块下跌的损失，最终就可以取得正收益。

轮动收益=轮动频率*轮动波幅

A股市场小市值股票之所以存在超高的轮动收益，很大程度上是来源于市场中大量的非理性散户行为，比如借壳、重组、题材概念、还有完全没道理的高送配等等，但随着市场环境的变化以及监管力度的加强，小市值股票短期内暴涨暴跌的现象肯定会逐渐受到遏制，个股波动的频率和波幅会日趋减小，必然会影响小市值策略未来的预期收益率。

简而言之，小市值轮动策略不抗跌。
